## 依赖包

bodmer/TFT_eSPI@2.0.0  https://github.com/Bodmer/TFT_eSPI

这个包的版本已经无法自动下载了，已放在assets文件夹下

使用方法: 把`TFT_eSPI@2.0.0.7z`解压到你的 %USERPROFILE%/.platformio/lib 目录下即可 （没有lib就新建）
